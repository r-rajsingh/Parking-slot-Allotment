RUNNING INSTRUCTIONS:

Step 1: Install Python virtual environment on your system using "pip install virtualenv".
Step 2: Create virtual environment using "virtualenv squadVenv".
Step 3: Activate virtual environment using "source squadVenv/bin/activate".
Step 4: Install other dependencies using "pip install -r requiremnets.txt".
        (Make sure you are in the same folder as "requirements.txt")
Step 5: Set up database configurations in "app.py" from line 8 to line 11.
Step 6: Place "input.txt" and "app.py" in the same folder.
Step 7: Run the program using "python app.py".
